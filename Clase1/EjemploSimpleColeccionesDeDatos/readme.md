# Proyecto: Ejemplo simple de colecciones de Datos en .NET


## Descripción del ejercicio

El programa realiza las siguientes acciones:

1. Declara una **lista genérica de strings** que representa colores
2. Agrega elementos iniciales y nuevos valores ingresados por el usuario
3. Muestra los valores almacenados en consola
4. Ordena la lista alfabéticamente
5. Inserta un elemento en una posición específica
6. Elimina una ocurrencia de un elemento
7. Introduce el uso de una **pila (`Stack<T>`)**


## Conceptos clave reforzados

- `List<T>` permite inserciones, eliminaciones y ordenamientos dinámicos
- `Sort()` organiza los elementos según el comparador por defecto
- `Insert(index, value)` permite control total sobre la posición
- `Stack<T>` sigue el principio **LIFO** (Last In, First Out)
- El uso de métodos auxiliares mejora la legibilidad del código
