# Ejercicio 5: Turnos de atención

## Objetivo
Utilizar la colección `Queue<T>` para aplicar el principio FIFO (First In, First Out).

## Enunciado
Simula una fila de atención utilizando una cola, donde las personas son atendidas en el mismo orden en que llegan.

La simulación debe permitir las siguientes acciones:

- Agregar personas a la fila de atención.
- Atender a una persona, removiéndola de la fila según el orden de llegada.
